Projeto Cordel Moderno

Site inspirado na literatura de cordel, combinando design minimalista, boa legibilidade e valorização da cultura popular em formato digital.

Saiba mais...

O Projeto Cordel Moderno foi criado para reinterpretar a tradicional literatura de cordel em um ambiente digital acessível e elegante. Utilizando HTML e CSS, o projeto priorizou um layout limpo, tipografia legível e fundos otimizados, reforçando o contraste visual entre texto e imagem. A proposta foi unir tradição e modernidade, mantendo a essência poética e visual do cordel, mas adaptada a um formato web responsivo e contemporâneo. O resultado é uma página leve, culturalmente rica e tecnicamente estruturada segundo boas práticas de desenvolvimento.

Projeto Android

Uma página temática dedicada ao sistema Android, desenvolvida com HTML semântico e design responsivo para oferecer uma experiência visual moderna e intuitiva.

Saiba mais...

O Projeto Android teve como objetivo explorar os fundamentos do desenvolvimento web utilizando HTML e CSS, com foco em estrutura semântica e boas práticas de layout. Durante o desenvolvimento, foram aplicadas técnicas de tipografia personalizada, otimização de imagens e alinhamento visual, garantindo leveza, clareza e acessibilidade. Projeto Rede Social

Simulação de uma interface mobile com ícones interativos de redes sociais, unindo design moderno e responsividade para reproduzir a experiência real de um smartphone

Saiba mais...

O Projeto Rede Social foi desenvolvido com o propósito de aprimorar a criação de interfaces digitais realistas, focando na simulação de um dispositivo móvel. A estrutura foi montada com HTML e CSS, destacando a organização de assets e o uso de elementos visuais que remetem a ícones de aplicativos populares. Além do apelo visual, o projeto reforçou conceitos de UX/UI, hierarquia visual e estética moderna, resultando em uma interface envolvente e intuitiva.
